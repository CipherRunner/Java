Есть Два типа сущностей

Человек с полями:
1. ID
2. Имя
3. Пол
4. Подразделение(Сущность подразделение)
5. Зарплата
6. Дата рождения

Подразделение с полями:
1. ID(генерится в програме)
2. Название

Дан CSV файл(прилагается в файлах к лабе), который содержит в себе информацию о людях. Нужно считать данные о людях из этого файла в список
В этой задаче нужно пользоваться встроенными Java  коллекциями

Для работы с CSV файлом рекомендую использовать библиотеку opencsv
(НО можете и без нее - это на ваше усмотрение)

Ее можно либо скачать в виде jar  файла и подключить к проекту если
не используете maven, либо подключить как maven зависимость

csvFilePath - путь к файлу. Файл закидывайте в ресурсы.
Если не получится, то укажите уж тогда полный путь.
separator - разделитель. В нашем случае  - ';'

Т.е. на выходе у вас должен получиться объект типа List в котором будут находится люди из файла.
